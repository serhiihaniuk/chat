Segmented is a radio-like row built on Base UI `ToggleGroup` + `Toggle`. The value is held as
a one-item array; each item flexes to an equal share. The active item is expressed through the
`pressed:` variant — fill **and** shadow, never colour alone — so exactly one item looks
raised.

## Usage

```tsx
import { Segmented, type SegmentedItem } from '@side-chat/side-chat-widget/ui/segmented';

const items: SegmentedItem[] = [
  { id: 'off', label: 'Off', Icon: Zap },
  { id: 'auto', label: 'Auto', Icon: Gauge },
  { id: 'max', label: 'Max', Icon: Sparkles },
];

<Segmented items={items} value={level} onValueChange={setLevel} />
```

Pass `stacked` to put the icon above the label (taller items), as the thinking-level control
does. Roving-tabindex keyboard nav is automatic via Base UI.
