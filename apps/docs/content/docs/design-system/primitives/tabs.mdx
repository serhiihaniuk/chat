Tabs is a Base UI `Tabs` composition with real panels — the distinction from Segmented,
which has none. `Tabs.Root` owns `value`/`onValueChange`; the `Tabs.List` and every
`Tabs.Panel` are siblings sharing one set of values, so a single array drives both places
and adding a group appears in each. The active trigger is expressed through the `selected:`
variant (`selected:bg-sidebar-accent` + `selected:text-foreground`); `hover:bg-accent` is
allowed here because the trigger mirrors the contract snippet.

## Usage

```tsx
import { Tabs } from '@base-ui/react/tabs';

<Tabs.Root value={value} onValueChange={setValue}>
  <Tabs.List>
    {groups.map((g) => (
      <Tabs.Tab key={g.value} value={g.value}>
        {g.icon}
        <span className="truncate">{g.label}</span>
      </Tabs.Tab>
    ))}
  </Tabs.List>

  {groups.map((g) => (
    <Tabs.Panel key={g.value} value={g.value}>
      {g.content}
    </Tabs.Panel>
  ))}
</Tabs.Root>
```

Drive the list and the panels from the SAME array so the two never drift. Roving-tabindex
keyboard nav between triggers is automatic via Base UI.
