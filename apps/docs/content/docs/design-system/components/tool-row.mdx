import { ToolRowDemo } from '@/components/demos/tool-row';

A tool row is one compact line inside the Reasoning panel: a leading status glyph followed by the tool name as plain text — no pill, no wrench. The name renders as `text-foreground` at `text-sm`, weight 500; only the glyph carries colour. It is informational, not interactive, so it is a plain `div` with `data-slot="tool-row"` and a `data-state`. The `running` state spins a `Loader2` in `text-primary`, `success` shows a `Check` in `text-success`, and `error` shows a `TriangleAlert` tinted by the `sc-error-glyph` hook (destructive mixed into `muted-foreground`).

## Usage

```tsx
import { ToolRow } from '@side-chat/side-chat-widget/ui/tool-row';

<ToolRow name="search_web" state="running" />
<ToolRow name="read_file" state="success" />
<ToolRow name="run_tests" state="error" />
```
