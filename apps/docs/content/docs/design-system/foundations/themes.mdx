The widget ships **four light themes**. Graphite is the base contract and carries no
attribute, so it inherits the host page's light/dark. Sapphire, Sage, and Ocean are
light-only Tier-2 overrides written via `data-sidechat-theme` on the widget root.

## How a theme is applied

```tsx
<SideChatWidgetRoot theme="sage">
  {/* every descendant re-skins through inheritance */}
</SideChatWidgetRoot>
```

- **Graphite** — cool charcoal neutrals (hue 264, not pure grey). The default; responds to a
  host `.dark` class.
- **Sapphire** — deep navy primary on cool neutrals.
- **Sage** — forest-green primary on green-tinted neutrals.
- **Ocean** — calm blue primary on blue-tinted neutrals.

<Callout type="info">
  Dark mode applies to **Graphite only** — the named themes are deliberately light-only.
  Toggling **Dark** with Sage/Ocean active is a no-op because the theme override wins.
</Callout>
