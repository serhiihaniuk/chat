---

The UI is set in **Plus Jakarta Sans** (self-hosted, weights 200–800); code is **JetBrains
Mono**. Every size in the scale is paired with a line-height so vertical rhythm can't drift.

Settings can swap the widget root to the System stack or an IBM Plex Sans stack when that
font is available on the host system; the fallback remains self-hosted Jakarta.

## The scale

<TokenTable group="typography" />

Weights are limited to `--weight-normal` (400), `--weight-medium` (500), and
`--weight-semibold` (600). Headings use semibold; nothing is bolder.
