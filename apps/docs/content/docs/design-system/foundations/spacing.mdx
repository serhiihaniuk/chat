---
title: Spacing
description: One density lever drives every pad and gap.
---

There is no discrete `--space-1…16` scale in implementation code. Instead, every pad and gap
is a `calc()` over a single lever, **`--space-unit`**, bridged to Tailwind's `--spacing` so
both component tokens and Tailwind utilities respond together.

<DesignControls />

<Preview>
  <div style={{ display: 'flex', flexDirection: 'column', gap: 'calc(var(--spacing) * 3)' }}>
    {[1, 2, 3, 4, 6].map((n) => (
      <div key={n} style={{ display: 'flex', alignItems: 'center', gap: 'calc(var(--spacing) * 2)' }}>
        <code style={{ width: '5rem', fontSize: '0.75rem', color: 'var(--muted-foreground)' }}>×{n}</code>
        <div style={{ height: '0.75rem', width: `calc(var(--spacing) * ${n} * 4)`, background: 'var(--primary)', borderRadius: 'var(--radius-sm)' }} />
      </div>
    ))}
  </div>
</Preview>

Drag the **Density** control above between Compact / Cozy / Roomy and every bar (and every
real component on the other pages) re-scales from this one variable.
