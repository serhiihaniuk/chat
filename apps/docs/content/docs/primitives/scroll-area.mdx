title: Scroll area
description: A bounded scroll region built on Base UI, with an overlay scrollbar that fades when idle.
---

import { ScrollAreaDemo } from '@/components/demos/scroll-area';

The scroll area wraps a bounded, scrollable region — used for panels, never the chat log. It renders the full Base UI `ScrollArea` tree (Root → Viewport → Scrollbar → Thumb) with a vertical overlay scrollbar. The `className` you pass styles the Viewport (the scrolling surface), so callers give it the border, radius, and padding. The fade-when-idle behaviour lives entirely in CSS, keyed on Base UI's own `data-orientation` / `data-hovering` / `data-scrolling` attributes via the `data-slot` hooks on the scrollbar and thumb — nothing drives it from JSX. The thumb width is `--scrollarea-w`; its fill is `--scrollarea-thumb`.

## Usage

```tsx
import { ScrollArea } from '@side-chat/side-chat-widget/ui/scroll-area';

<ScrollArea className="rounded-lg border border-border">
  <div className="flex flex-col gap-2 p-3">{/* bounded content */}</div>
</ScrollArea>
```
