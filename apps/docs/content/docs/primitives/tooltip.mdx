Tooltip is a direct Base UI `Tooltip` composition — `Tooltip.Provider`, `Root`, `Trigger`, `Portal`, `Positioner`, and `Popup` — used to label the header icon buttons (Settings / New chat / Close) in place of the native `title` attribute. A single `Tooltip.Provider` near the root sets one shared open `delay`, and every icon button without a visible label pairs its tooltip with an `aria-label`. The popup has no dedicated palette: it carries the `tooltip-content` slot and inherits the menu surface (`--popover`, `--popover-foreground`, `--border`, `--shadow-popover`), and is portaled into the widget root via `usePortalContainer()` so it escapes overflow clipping while staying inside the themed subtree.

## Usage

```tsx
import { Tooltip } from '@base-ui/react/tooltip';

<Tooltip.Provider delay={500}>
  <Tooltip.Root>
    <Tooltip.Trigger
      render={<button type="button" className="sc-icon-btn" aria-label="Settings">…</button>}
    />
    <Tooltip.Portal container={container}>
      <Tooltip.Positioner sideOffset={6}>
        <Tooltip.Popup data-slot="tooltip-content">Settings</Tooltip.Popup>
      </Tooltip.Positioner>
    </Tooltip.Portal>
  </Tooltip.Root>
</Tooltip.Provider>
```
