Row is a className pattern, not its own Base UI part: a selectable line is
`[leading media?] [title + optional subtitle] [trailing check/indicator?]`, and consumers
inline `rowBaseClass` then append a form-specific state variant.

It appears in two forms: as the styled content of a Base UI `Select.Item` / `Menu.Item` /
`Combobox.Item`, where the active state is `highlighted:` and the trailing check is
`selected:` on the `ItemIndicator`; or as a standalone `<button>` (the conversation row),
where the active state is the semantic `aria-current="true"`.

Both forms read `--row-bg-hover` for hover/highlight and `--row-bg-active` for selected
standalone rows. Those tokens resolve to `--accent`, so the docs Accent surface control
re-skins picker rows, sidebar conversation rows, user message bubbles, and other row states
together.

Two rules are load-bearing in both forms: the title column carries `min-w-0` and the text
carries `truncate`, or a long title pushes the whole panel wider; the trailing indicator
always sits in the DOM at `opacity-0` and is only revealed by state, so selection never
shifts the layout.

## Usage

```tsx
import { rowBaseClass } from '@side-chat/side-chat-widget/ui/row';

// A) As a Base UI item - active via highlighted:, check via selected:
<Select.Item value={m.id} className={cn(rowBaseClass, 'highlighted:bg-(--row-bg-hover)')}>
  <span className="sc-media">{m.icon}</span>
  <span className="flex min-w-0 flex-col">
    <Select.ItemText className="truncate text-sm font-medium text-foreground">{m.name}</Select.ItemText>
    <span className="truncate text-xs text-muted-foreground">{m.desc}</span>
  </span>
  <Select.ItemIndicator className="ml-auto flex shrink-0 text-primary opacity-0 selected:opacity-100">
    <Check className="size-4" />
  </Select.ItemIndicator>
</Select.Item>

// B) As a standalone conversation button - active via aria-current
<button
  type="button"
  aria-current={isActive ? true : undefined}
  className={cn(rowBaseClass, 'hover:bg-(--row-bg-hover) aria-[current=true]:bg-(--row-bg-active)')}
>
  <span className="flex min-w-0 flex-col">
    <span className="truncate text-sm font-medium text-foreground">{title}</span>
    <span className="truncate text-xs text-muted-foreground">{when}</span>
  </span>
  <span className="ml-auto size-1.5 shrink-0 rounded-full bg-primary opacity-0 aria-[current=true]:opacity-100" />
</button>
```
