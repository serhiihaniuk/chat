A row of ghost icon+label buttons shown under a completed assistant answer. All
styling lives in the `sc-action` hook class — a transparent background, an
`--accent` hover, and a `&[data-copied]` success swap to `--success`. Pressing
Copy flips that same button to a transient "Copied" state for ~1.3s, then reverts;
Retry re-runs the turn.

## Usage

```tsx
import { MessageActions } from '@side-chat/side-chat-widget/ui/message-actions';

<MessageActions onCopy={() => copyAnswer()} onRetry={() => retryTurn()} />
```

Both `onCopy` and `onRetry` are optional. The "Copied" feedback is owned by the
component, so the parent only needs to perform the clipboard write.
