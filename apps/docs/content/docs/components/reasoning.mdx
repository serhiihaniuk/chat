Reasoning is a foldable thinking trace built on Base UI `Collapsible`. Its panel interleaves thought lines and `ToolRow` entries as siblings, in stream order — never a separate tool block below the answer. The header label shimmers with a subtle `animate-pulse` while the model is still thinking, and the `ChevronDown` rotates to follow the controlled open state. Open state is controlled, so a parent can expand live thinking, collapse completed minimal traces, or keep detailed traces open while leaving the trigger user-toggleable. The panel height animates from Base UI's exposed `--collapsible-panel-height` through the `sc-collapsible-panel` hook class.

## Usage

```tsx
import { Reasoning, type ReasoningItem } from '@side-chat/side-chat-widget/ui/reasoning';

const items: ReasoningItem[] = [
  { kind: 'thought', id: 't1', text: 'Reading the conversation context.' },
  { kind: 'tool', id: 'x1', name: 'search_files', state: 'success' },
];

<Reasoning
  items={items}
  label="Thought for 4s"
  thinking={false}
  open={open}
  onOpenChange={setOpen}
/>
```
